# char-image-model

## Setup

### uv

```bash
uv python install 3.12
uv venv
uv pip install \
    --upgrade pip \
    --no-cache-dir \
    -r requirements.txt
```

### pyvenv

```bash
python -m venv sandbox
```

有効化

```bash
source sandbox/bin/activate
```

```bash
pip install \
    --upgrade pip \
    --no-cache-dir \
    -r requirements.txt
```
